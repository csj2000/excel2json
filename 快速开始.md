# 🚀 快速开始指南

## 第一步：安装依赖

在项目根目录打开终端，运行：

```bash
npm install
```

⏱️ 这可能需要 2-5 分钟，具体取决于网络速度。

## 第二步：启动开发版本

安装完成后，运行：

```bash
npm run electron:dev
```

🎉 应用窗口将自动打开！

## 第三步：测试功能

1. **准备一个 Excel 文件**（.xlsx 格式）
2. **拖拽文件**到应用窗口，或点击"选择文件"按钮
3. **选择工作表**（左侧列表）
4. **查看数据预览**（右侧表格）
5. **选择 JSON 格式**
6. **点击"转换当前文件"**，选择保存位置
7. ✅ 完成！

## 第四步：打包为可执行程序（可选）

如果想要打包成独立的 .exe 程序：

```bash
npm run dist:win
```

📦 打包后的安装程序在 `dist/` 目录中。

---

## 常见问题

### Q1: npm install 报错
- 确保已安装 Node.js 18 或更高版本
- 尝试使用国内镜像：`npm config set registry https://registry.npmmirror.com`

### Q2: 应用无法启动
- 检查是否有其他程序占用 5173 端口
- 删除 `node_modules` 文件夹，重新运行 `npm install`

### Q3: 打包失败
- 确保已运行 `npm run build` 成功
- 检查磁盘空间是否充足（需要约 500MB）

### Q4: 想修改应用图标
- 将 .ico 文件替换到 `build/icon.ico`
- 重新运行 `npm run dist:win`

## 开发命令速查

| 命令 | 说明 |
|------|------|
| `npm install` | 安装依赖 |
| `npm run electron:dev` | 启动开发模式 |
| `npm run build` | 构建前端代码 |
| `npm run dist:win` | 打包 Windows 程序 |
| `npm run pack` | 打包但不安装 |
